# Wildlife Collision MLOps

A simple, beginner-friendly end-to-end MLOps project structure.

## Folders

- backend/: Python backend (FastAPI)
- frontend/: Placeholder for future React app
- data/: Data storage (raw, bronze, silver, gold)
- models/: Trained models
- logs/: Log files
- config/: Configuration files
